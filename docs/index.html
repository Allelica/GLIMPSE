Genotype Likelihoods IMputation and PhaSing mEthod

GLIMPSE is a phasing and imputation method designed for low-coverage sequencing datasets.

Add description of the main features of GLIMPSE.

GLIMPSE is developed by Simone Rubinacci and Olivier Delaneau at the University of Lausanne, Switzerland.

GLIMPSE tools is available under the MIT licence on the Github repository https://github.com/odelaneau/GLIMPSE.

GLIMPSE tools

GLIMPSE chunk

Defines chunks where to run imputation

GLIMPSE phase

Performs phasing and imputation refining genotype likelihoods

GLIMPSE ligate

Concatenate chunks ligating phased information

GLIMPSE sample

Generate haplotype calls by sampling haplotypes
